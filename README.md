# Webpack SCSS + HTML + Images Starter

Это минимальный шаблон для сборки верстки с помощью Webpack, поддержкой SCSS, оптимизацией изображений (webp, svg) и авто‑префиксинга.

Установить зависимости:

```powershell
npm install
```

Запуск в режиме разработки (локальный сервер, hot reload):

```powershell
npm run dev
```

Сборка продакшен‑версии (оптимизация CSS/JS/изображений):

```powershell
npm run build
```

Структура проекта:

- `src/` — исходники
  - `index.html` — шаблон HTML
  - `index.js` — точка входа (импортирует стили и ассеты)
  - `styles/main.scss` — основные SCSS
  - `assets/` — изображения, шрифты и т.п.
- `dist/` — автоматически создаётся Webpack'ом

Примечания:

- В продакшен‑режиме изображения будут оптимизированы и дополнительно конвертированы в `webp` (если это возможно) с помощью `image-minimizer-webpack-plugin` и плагинов `imagemin-webp`.
- Добавляйте свои изображения в `src/assets/` и импортируйте их в JS или используйте в CSS.
